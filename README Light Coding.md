## Light Coding
  Make your coding easier!
  Loadstring :
  ```lua
  loadstring(game:HttpGet("https://raw.githubusercontent.com/Ajkaejs/Fast_And_Simple_Scripts/main/Scripts/Light-Coding.lua"))()
  ```
 ### Functions
  say(text) - print text example :
  ```lua
  say("hi")
  -- Print hello to the console
  ```
  
  warnmsg(text) - warn message example :
  ```lua
  warnmsg("Warning")
  -- Print ⚠️ Warning to the console
  ```
  
  err(text) -- WIP
  ```lua
  err("Error")
  -- Print ❌ Error to the console
  ```
  
  service(NameService) -- get service example :
  ```lua
  service(game.Players)
  -- get Players Service
  ```
  
  create(NameClass) - Creates an object with the class name example :
  ```lua
  create("Part") -- Creates Part
  ```
  
  pfireclickd(pathToObject) - Fire Click Detector need object path
  ```lua
  pfireclickd(workspace.ClickDetector)
  -- fire click detector in workspace
  ```
  
  pfiretouchintrest(object) - Fire Touch Interest (Touch Transmitter) - need object path
  ```lua
  pfiretouchintrest(workspace.Part.Touchintrest)
  -- Fire Touch Interest in workspace.Part
  ```

  pfireproxpromt(object) -- Fire Proximity Prompt need object path
  ```lua
  pfireproxpromt(workspace.ProximityPrompt)
  -- fire proximity prompt in workspace
  ```
